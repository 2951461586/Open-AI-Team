# Electron

This directory is a **secondary app shell** for desktop packaging/experimentation.

It is not part of the primary first-time onboarding path for AI Team Runtime.

## Role

- desktop shell / packaging surface
- secondary to the main runtime + dashboard web story
- may be experimental depending on current repository phase

## Contribution rule

Do not treat `electron/` as equivalent in authority to:
- `dashboard/`
- `apps/api-server/`
- `packages/agent-harness/`
- `packages/team-runtime/`

If the desktop shell becomes a first-class product surface later, it should get explicit docs and release rules.
