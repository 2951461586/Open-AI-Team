# Background References

> Third-party or extracted reference material kept for comparison, research, and context.

## What lives here

This directory may contain:
- extracted guides from external products/services
- comparison references used during design or cleanup work
- background material worth preserving for context

## Boundary

`references/` is **background context**, not product authority.

Treat it as:
- useful for comparison and research
- non-primary for contributors
- not a replacement for repository architecture / release / boundary docs

If a file here conflicts with current repository docs, prefer:
- `../README.md`
- `../GETTING-STARTED.md`
- `../ARCHITECTURE.md`
- `../docs/architecture/`

## One-line rule

References can inform decisions, but they do not define the product boundary of this repository.
