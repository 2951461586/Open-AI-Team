# Public Example Configs

These files are the public-safe config side of the **Team Runtime public example track**.

See also:
- `../../examples/team-runtime-public/README.md`
- `../../GETTING-STARTED.md`

These files are neutral examples for external users and contributors.

They are intentionally separate from the active repository runtime inventory under `config/team/`, which may still contain operator-oriented deployment details for current maintainers.

Use these example files when:
- onboarding a new external contributor
- documenting public setup
- preparing a fork or independent deployment

Do not treat them as live production state.
