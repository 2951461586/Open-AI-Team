1. [fact] MuleRun JWT 自动化管理调研结论 (2026-03-30):
- JWT 类型: HS256, issuer=mulerun.com, audience=user
- JWT 有效期: 7天 (iat→exp)
- Dashboard API base: https://mulerun.com (不是 api.mulerun.com)
- 有效的 dashboard 端点: /user/info (200), /user/balance (200), /user/wallets (200), /user/credit-usage (200)
- 整个 /api/v1/* 路由被 JWT 中间件保护，没有公开的无认证登录/注册/refresh API
- 登录使用 Cloudflare Turnstile 反bot + 前端 SPA 渲染
- 没有发现 JWT refresh 端点
- 结论: 无法通过纯 API 调用实现 JWT 自动续期，只能通过:
  1. 半自动: 人工登录粘贴 JWT（当前方案）
  2. 全自动: headless browser 模拟登录（需过 Turnstile，风险高）
  3. 最佳实践: 提前告警 JWT 过期 + 简化人工更新流程 — status=inferred
