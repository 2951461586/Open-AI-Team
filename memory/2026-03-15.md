### 5. 配置工具经验

- 本地 `scripts/openclaw-config-lint.py` 当前会把 `models.providers.*.apiKey` 误判为必须是字符串；但生产配置实际已使用对象形式的 secrets 引用（`source=file, provider=filemain, id=/...`）。
- 因此后续做 OpenClaw provider 接入验证时，**不能只依赖该 lint 脚本**，应以：
  - JSON 可解析；
  - gateway 可正常启动；
  - provider / model 在运行时可用；
  作为更可信的验收标准。
