## 你下一次抓包要抓到的“关键请求”长什么样
在 DevTools → Network 里，它通常会满足几个特征（至少中两条）：

1) **类型**：Fetch/XHR 或 WS  
2) **响应**：`text/event-stream`（流式）/ 或一个明显的 JSON（非流式）  
3) **耗时**：明显更久（因为在等模型回复），Size 也更大

5) **域名不一定只在 copilot.microsoft.com**：企业/工作账号场景，可能会出现 `*.office.com`、`graph.microsoft.com`、`substrate.office.com` 等（所以别只盯 copilot 那个域）
